# 🛍️ Fashion Shop

A modern, responsive fashion e-commerce website built with HTML, CSS, JavaScript, and Bootstrap.

## 🚀 How to Open

### Option 1: Open Directly (No Setup)
1. Download or clone this repository
2. Double-click `index.html` to open in your browser

### Option 2: Live Server (Recommended)
1. Install VS Code extension: **Live Server**
2. Right-click on `index.html` → **Open with Live Server**

### Option 3: Python HTTP Server
```bash
python -m http.server 8000
# Then open: http://localhost:8000
```

## ✨ Features
- 🔥 Hero section with scroll animations
- 🏷️ Shop by Category
- ⚡ Flash Sale with countdown timer
- 🆕 Trending Now products
- 🏢 Brands marquee
- 💬 Customer testimonials
- 🛒 Shopping cart functionality
- 📱 Fully responsive design

## 🎨 Tech Stack

- CSS3 (with animations & clip-path)
